# Anti-Money Laundering (AML) — Graph Analysis & GNN

Detect money laundering patterns in transaction networks:
- **Graph construction**: accounts as nodes, transactions as edges
- **Graph features**: PageRank, betweenness centrality, clustering coefficient
- **GraphSAGE** (Graph Neural Network) for node classification
- AML pattern detection: fan-out, fan-in, cycles, scatter-gather
